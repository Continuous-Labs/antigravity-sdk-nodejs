# Google Antigravity SDK for Node.js

A high-performance, robust, and highly idiomatic Node.js/TypeScript SDK for building autonomous AI agents and multi-agent cascades using Gemini models.

This project is a direct port based on the official [Google Antigravity SDK for Python](https://github.com/google-antigravity/antigravity-sdk-python.git), maintaining 1:1 conceptual parity with its Layer 1 (`Agent`), Layer 2 (`Conversation`), Hooks, Policies, Tools, and Triggers systems.

---

## Installation & Setup

> [!IMPORTANT]
> **Recommended Package Manager**: We strongly recommend using **`pnpm`** instead of `npm` or `yarn` for installation and development. `pnpm` provides enhanced security through a content-addressable store and strict, non-flat `node_modules` structures which prevent phantom dependency security holes.

### 1. Install Dependencies
Initialize and add the SDK to your project using `pnpm`:

```bash
pnpm add google-antigravity
```

### 2. Set your Gemini API Key
The localharness connection strategy requires a Gemini API key. Set it as an environment variable:

```bash
export GEMINI_API_KEY="your-api-key-here"
```
*(If you need an API key, you can get one from [Google AI Studio](https://aistudio.google.com/app/api-keys)).*

## Quick Start (Hello World)

Here is how to set up a basic agent and perform a simple conversational turn using the safe, context-managed `Agent.run` pattern:

```typescript
import { Agent, LocalAgentConfig } from "google-antigravity";

async function main() {
  const config = new LocalAgentConfig({
    model: "gemini-3.5-flash",
  });

  console.log("Starting Antigravity Agent...");
  await Agent.run(config, async (agent) => {
    console.log("Sending prompt: 'Hello! Who are you?'");
    const response = await agent.chat("Hello! Who are you?");
    const text = await response.text();
    console.log("Agent Response:", text);
  });
}

main().catch(console.error);
```

## Core Capabilities

### 1. Real-Time Token Streaming
The SDK allows you to stream both the agent's internal reasoning (thoughts) and conversational text tokens as they are generated by the model in real-time:

```typescript
await Agent.run(config, async (agent) => {
  const response = await agent.chat("Write a short poem about antigravity.");

  // Stream reasoning/thoughts in real-time
  const thoughts = response.thoughts;
  for (let res = await thoughts.next(); !res.done; res = await thoughts.next()) {
    process.stdout.write(res.value);
  }

  // Stream conversational tokens
  for await (const chunk of response) {
    process.stdout.write(chunk);
  }
});
```

### 2. Custom Host-Side Tools
Equip your agents with custom JavaScript/TypeScript functions that they can invoke dynamically:

```typescript
// Custom JavaScript function tool
async function get_weather_forecast(args: { location: string }) {
  return {
    location: args.location,
    temperature: "22 C",
    forecast: "Sunny"
  };
}

const config = new LocalAgentConfig({
  model: "gemini-3.5-flash",
  tools: [get_weather_forecast]
});
```

### 3. Safety Policies & Scoping
Strictly govern tool execution using priority-based safety rules and directory-scoping policies:

```typescript
import { policy } from "google-antigravity";

const config = new LocalAgentConfig({
  model: "gemini-3.5-flash",
  policies: [
    // Block dangerous run_command tools entirely
    policy.deny("run_command"),
    // Confine all file reads and writes strictly to the configured workspaces
    ...policy.workspaceOnly([process.cwd()])
  ]
});
```
